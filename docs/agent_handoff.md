# AI Handoff Document: "Data-Dex: Ultimate Stat Lab"

## 1. Project Overview & Constraints

* **Application Name:** Data-Dex: Ultimate Stat Lab
* **Target Audience:** 3rd–5th graders (ages 8–11). Sessions are 20–30 minutes.
* **Objective:** Teach foundational data visualization (Radar, Bar, Scatter) using Pokémon data.
* **Tech Stack:** Python, Dash, Plotly, Dash Mantine Components (DMC), `pandas`, `pyarrow`.
* **Key Constraints:** * Must load instantly on classroom tablets (Network/API latency during runtime is unacceptable).
* Dash <= 2.17 requires `dash._dash_renderer._set_react_version("18.2.0")` for DMC compatibility.
* Simplified, action-oriented language (e.g., "Energy Blast" instead of "Special Attack").

## 2. Architecture & File Structure

The project uses a modular structure, serving a highly compressed, strictly typed local Parquet file to ensure instant read times.


### C. Data Management (`data_manager.py`)

Loads the Parquet file, applies kid-friendly translations, and generates dynamic official artwork URLs.

```python
import pandas as pd

def load_and_clean_data(filepath="data/pokemon.parquet"):
    df = pd.read_parquet(filepath, engine='pyarrow')
    
    rename_map = {
        'Sp. Atk': 'Energy Blast',
        'Sp. Def': 'Energy Shield',
        'Type 1': 'Primary Type',
        'Type 2': 'Secondary Type'
    }
    df = df.rename(columns=rename_map)
    df['Secondary Type'] = df['Secondary Type'].fillna('None')
    
    if '#' in df.columns:
        df['Image_URL'] = df['#'].apply(
            lambda x: f"https://raw.githubusercontent.com/PokeAPI/sprites/master/sprites/pokemon/other/official-artwork/{x}.png"
        )
    return df

```

### D. Radar Chart Logic (`visualizations.py`)

Builds a Plotly `Scatterpolar` chart with a **static radial axis** (`[0, 160]`) so polygon shapes are mathematically comparable. Closes the polygons and uses smooth cubic-in-out transitions.

```python
import plotly.graph_objects as go
import pandas as pd

def create_radar_chart(df: pd.DataFrame, pokemon_names: list) -> go.Figure:
    categories = ['HP', 'Attack', 'Defense', 'Speed', 'Energy Shield', 'Energy Blast']
    fig = go.Figure()

    if not pokemon_names or df.empty:
        fig.update_layout(polar=dict(radialaxis=dict(visible=True, range=[0, 160])), showlegend=False, paper_bgcolor="rgba(0,0,0,0)", plot_bgcolor="rgba(0,0,0,0)")
        return fig

    selected_data = df[df['Name'].isin(pokemon_names)]

    for index, row in selected_data.iterrows():
        stats = [row[cat] for cat in categories]
        stats.append(stats[0]) # Close the polygon
        closed_categories = categories + [categories[0]]

        fig.add_trace(go.Scatterpolar(
            r=stats, theta=closed_categories, fill='toself', name=row['Name'],
            hoverinfo="text", text=[f"{cat}: {val}" for cat, val in zip(closed_categories, stats)]
        ))

    fig.update_layout(
        polar=dict(
            radialaxis=dict(visible=True, range=[0, 160], showticklabels=False, gridcolor="lightgrey"),
            angularaxis=dict(tickfont=dict(size=14, color="darkslategray"), rotation=90, direction="clockwise")
        ),
        showlegend=True,
        legend=dict(orientation="h", yanchor="bottom", y=-0.2, xanchor="center", x=0.5),
        margin=dict(l=40, r=40, t=40, b=40),
        transition=dict(duration=500, easing="cubic-in-out"),
        paper_bgcolor="rgba(0,0,0,0)",
    )
    return fig

```
